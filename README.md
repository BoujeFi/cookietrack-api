# CookieTrack API

The API that powers CookieTrack.

---

## Endpoints
- **/avalanche/wallet?address=\<wallet_address\>** - Returns a wallet's token balances on the Avalanche Chain.

---

## Local Development:

Uncomment line 26 on `functions/index.js`.

```
npm i
cd functions
npm i
cd..
npm start
```
